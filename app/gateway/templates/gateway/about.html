{% translate "The safety pipeline" %}

  1. {% translate "Read only" %} — {% translate "The worker connects with certificate-verified IMAPS, opens INBOX read-only, and fetches with BODY.PEEK. It has no SMTP implementation." %}
  2. {% translate "Bounded processing" %} — {% translate "MIME and HTML are parsed with size and part limits. Active HTML, remote images, and raw message bytes are not exposed." %}
  3. {% translate "Signals, not trust" %} — {% translate "SPF, DKIM, DMARC, and ARC provider headers are treated as unverified claims and never approve a message automatically." %}
  4. {% translate "Owner decision" %} — {% translate "Every new message is quarantined. Only the owner can approve the sanitized representation." %}
  5. {% translate "Restricted agent API" %} — {% translate "The API is GET-only and returns approved sanitized fields. It exposes no IMAP, SMTP, delete, move, reply, raw-source, or attachment-download capability." %}

{% translate "Trusted authserv IDs — advanced and optional" %}

{% translate "An authserv-id is the identifier before the first semicolon in an Authentication-Results header added by the receiving provider:" %}

Authentication-Results: mx.example.test; dmarc=pass; dkim=pass
                        ^ authserv-id

{% translate "It is not necessarily the IMAP hostname, sender domain, or email domain. Do not paste the complete header. Leave the field empty unless your provider documents the ID or you have verified it independently across known messages." %}

{% translate "Even a matching ID can be forged inside email content and never causes automatic approval." %}

{% translate "Attachments and PDFs" %}

{% translate "MailGate records only attachment filename, type, size, hash, and danger flags. Attachment bytes are discarded and never returned to the agent API. PDF content is currently not interpreted; it is contained, not declared safe." %}

{% translate "What MailGate stores" %}

{% translate "The active database stores encrypted mailbox credentials, sanitized message text, technical signals, decisions, minimal attachment metadata, tokens as hashes, and audit events. Telemetry is disabled and there is no MailGate cloud service." %}
